## MySql数据类型
> [参考链接](https://www.runoob.com/mysql/mysql-data-types.html)
> MySQL 中定义数据字段的类型对你数据库的优化是非常重要的。  
> MySQL 支持多种类型，大致可以分为三类：数值、日期/时间和字符串(字符)类型。

### 数值类型
- 整型
    - tinyint
    - smallint
    - mediumint
    - int / integer
    - bigint
- 浮点型
    - float
    - double
    - decimal


### 日期和时间类型
- DATE
- TIME
- YEAR
- DATETIME
- TIMESTAMP

### 字符串类型
- 字符串
    - CHAR
    - VARCHAR
    - BLOB
    - TEXT
    - MEDIUMTEXT
    - LONGTEXT
- 二进制
    - TINYBLOB
    - TINYTEXT
    - MEDIUMBLOB
    - LONGBLOB
