# R 语言基础

## 数据结构/基本运算
- R语言的数据类型主要有
    - 数值型
    - 逻辑型 `TRUE/FALSE`
    - 字符型
    - 复数型
    - 原始型（二进制）
    - 缺省值
        - `is.na(待识别值)` 判断是否缺失值
- 数据对象
    - 向量（vector）
    - 矩阵（matrix）
    - 数组（array）
    - 因子（factor）
    - 列表（list）
    - 数据框（data frames）

- 向量函数
    - 生成函数 `c("元素1","元素2","...")` `c(1,2,3)`
    - 连接函数 `paste("自变量",切片区域,sep="连接符")`
        - `paste("x", 1:5, sep="")`: 将字符变量`"x"`与`"1"~"5"`进行连接
        - 连接结果得到`"x1" "x2" "x3" "x4" "x5“`
    - 赋值函数 `assign("目标向量名",值)`
        - `assign("w",c(1,3,5,7,9))` 则 `w -> c(1,3,5,7,9)`
    - 运算函数
        - 使用 `*`/`/`/`^` 对分量运算
        - 使用 `%/%` 进行整数除法（不保留余数）
        - 使用 `%%` 取除法余数
    - 正则生成 `:`
        - 使用 `大数:小数` 生成逆向序列
        - 使用 `小数:大数` 生成正向序列
    - 有序序列 `seq(起始值,终止值,步长)`
        - `seq(起始值, by=步长, length=生成元素数)`
    - 重复生成 `rep(重复对象,重复次数)`
    - 其他函数
        - 函数length()可以返回向量的长度
        - mode()可以返回向量的数据类型
        - min()返回向量的最小值，max()返回向量的最大值
        - range()返回向量的范围
        - which.min()、which.max()返回在第几个分量求到最小、最大值。

## 第三次作业

### 题干
1. 研究教材56页（章节4.1.2）由均匀分布随机数经变换产生标准正态分布随机数的方法：
    - 模拟图4-2的随机数生成及绘图效果
    - 比较不同的n值情形下，理论分布与经验分布之间的差异程度

2. 根据教材60页（章节4.1.5）中表4-1和表4-2：
    - 应用自己学号的后两位做随机数的种子，分别生成50个以上的贝塔分布、伽玛分布、威布尔分布的随机数
    - 参数自由设定
    - 比较理论分布与样本经验分布之间的差异程度

### 答案

### 涉及知识

使用 `runif(n,  min=0,  max=1)` 生成均匀分布平均数

假设 $U_1$ 和 $U_2$ 是 $[0,1]$ 上均匀分布随机数，做如下变换
$$
X_1=\sqrt{-2\log{\left(U_1\right)}}\cos{\left(2\pi U_2\right)},X_2=\sqrt{-2\log{\left(U_1\right)}}\sin{\left(2\pi U_2\right)}
$$
这样得到的随机数独立同分布于 $N\left(0,1\right)$。
